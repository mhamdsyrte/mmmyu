# File Manager Pro

تطبيق إدارة ملفات لأندرويد مبني بـ **C# (.NET MAUI)** مع واجهة **HTML/CSS عبر Blazor Hybrid**.

## المزايا
- تصفح كامل للتخزين الداخلي والمجلدات (تنزيلات، كاميرا، مستندات، تخزين خارجي إن وجد)
- نسخ / قص / لصق / حذف / إعادة تسمية (لعنصر واحد أو تحديد متعدد)
- إنشاء مجلدات وملفات نصية جديدة
- محرر نصوص مدمج (يفتح .txt, .md, .json, .cs, .js, .html, .css... إلخ) مع مؤشر تغييرات غير محفوظة
- تصميم داكن مستوحى من أدوات الطرفية (Terminal) مع شريط مسار يشبه سطر الأوامر

## كيف تبني ملف APK (بدون أي إعداد على جهازك)

1. أنشئ مستودع (repository) جديد وفارغ على GitHub.
2. ارفع كل ملفات هذا المشروع إليه (git push أو السحب والإفلات عبر واجهة GitHub في المتصفح).
3. بمجرد الرفع، سيعمل **GitHub Actions** تلقائيًا (الملف موجود في `.github/workflows/build-android.yml`) ويبني التطبيق — يستغرق عادة 8-12 دقيقة.
4. اذهب لتبويب **Actions** في المستودع → افتح آخر عملية تشغيل (run) → بعد اكتمالها بنجاح، ستجد **Artifacts** في الأسفل باسم `FileManagerPro-apk`.
5. حمّل الملف، فك الضغط عنه (zip)، وثبّت `FileManagerPro.apk` على هاتفك (فعّل "تثبيت من مصادر غير معروفة" إن طُلب منك ذلك).

## صلاحيات مهمة
عند أول تشغيل، سيطلب منك التطبيق صلاحية "الوصول لكل الملفات" (All files access) — هذه ضرورية لأي تطبيق إدارة ملفات حقيقي مثل هذا حتى يقدر يتصفح كامل التخزين، وافق عليها من شاشة الإعدادات التي ستفتح تلقائيًا.

## هيكل المشروع
```
Components/Main.razor      → كل واجهة ومنطق التطبيق (Blazor)
Services/FileService.cs    → عمليات نظام الملفات (نسخ/نقل/حذف/قراءة/كتابة)
Platforms/Android/         → صلاحيات وإعدادات أندرويد الأساسية
wwwroot/css/app.css        → التصميم الكامل
.github/workflows/         → البناء التلقائي عبر GitHub Actions
```

## تعديل الاسم أو الأيقونة
- الاسم: غيّر `ApplicationTitle` و `ApplicationId` في `FileManagerPro.csproj`.
- الأيقونة: استبدل ملفي SVG في `Resources/AppIcon/`.
